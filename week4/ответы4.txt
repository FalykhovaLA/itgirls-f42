1.Встроенные стили - использование атрибута style в начальном теге HTML (<a style="font-color: red;" href="#">Ссылка</a>).
Внутренние таблицы стилей - использование элемента <style> в разделе заголовка документа <head>.
Внешние таблицы стилей - Использование элемента <link>, указывающего на внешний файл CSS (наиболее предпочтительный вариант)

2.обеспечивает для HTML-элементов лучшую кроссбраузерность в стилях по умолчанию

3.Это специальные конструкции, которые начинаются с символа @. Директивы чаще всего влияют на весь документ, но сами по себе ничего не стилизуют

4.Margin - это отступ за пределами блока (внешний отступ), а padding - отступ внутри блока (внутренний оступ).

5.Селектор,обладающи большим значением специфичности, обладает большим приоритетом.
6.различие между CSS и CSS3 состоит в том, что CSS — это простой язык дизайна, который позволяет создавать привлекательные веб-страницы,
 тогда как CSS3 — это последняя третья версия языка каскадных таблиц стилей, имеющая новые функции, позволяющие работать с веб-дизайном намного более эффективно.
7.Псевдоклассы – это селекторы, которые определяют состояние уже существующих элементов, которое может меняться при определенных условиях.
Псевдоэлементы – это селекторы, которые определяют область элементов, которая изначально отсутствует в дереве документа. 
Эта область создается искусственно с помощью CSS.
9.Правило @font-face src позволяет задать название локального шрифта, т.е. если у пользователя на компьютере уже установлен нужный шрифт, то будет использоваться именно он.
